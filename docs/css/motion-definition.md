# Css 动画定义

## 动画样式

- `Fade` 渐入/渐出
- `Slide` 滑入/滑出
- `Focus` 聚焦
- `Blur` 模糊
- `Bounce` 反弹
- `Expand` 扩展
- `Shrink` 收缩
- `Pop` 弹出
- `Compress` 压缩
- `Shake` 摇摆
- `Confetti` 彩带
- `Typewriter` 打字机模式

## 动画附加条件

- `delay` 延时
- `duration` 时长
- `translate-x, translate-y` 平移
- `scale` 尺寸
- `rotate` 旋转
- `in` 进入动画
- `out` 离开动画
