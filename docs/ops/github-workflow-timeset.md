# Github Actions 中的 工作流定时执行

```yml
on:
  schedule:
    - cron: '30 5,17 * * *'
```

`cron` 包含五个字段，分别是:
- 分钟
- 小时
- 日期
- 月份
- 星期

可使用的特殊字符:
- \* 表示任意值
- , 用于分割多个值
- \- 表示范围
- / 用于指定间隔

示例：

```txt
0 1 * * * # 每天 UTC 1:00 AM (北京时间 9:00 AM) 运行
0 0 * * 0 # 每周日 UTC 0:00 运行
*/15 * * * * # 每 15 分钟运行一次
```
